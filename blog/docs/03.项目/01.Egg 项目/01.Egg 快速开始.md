## [快速开始](https://eggjs.org/zh-cn/intro/quickstart.html)
```sh
# 新建文件夹[项目名]
mkdir api && cd api
# 利用手脚架快速生成
npm init egg --type=simple
# 安装依赖
npm install
# 启动项目
npm run dev
open http://localhost:7001
```
